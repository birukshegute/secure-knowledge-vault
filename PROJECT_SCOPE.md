# Project Scope

## Objective
Build and deploy a secure Django-based web application on AWS to demonstrate
backend development, cloud infrastructure, and cybersecurity skills.

## In Scope (v1)
- Django backend with authentication
- Secure note CRUD
- Encrypted data storage
- AWS EC2 deployment
- S3-based file storage
- One asynchronous background task
- Basic logging and monitoring
- Manual security testing and remediation

## Out of Scope
- Mobile applications
- Public SaaS launch
- Payments or subscriptions
- High availability or auto-scaling
- Advanced SIEM integrations

## Definition of Done
- Application deployed on AWS and reachable publicly
- At least one successful attack simulated
- Vulnerabilities patched and documented
- Infrastructure destroyed after completion
